# snack-time

An app solution for small restaurants to delivery snacks and order made by whatsapp

Using:

-   React Native Expo [**Framework**]
-   Expo router [**Route**]
-   TypeScript [**Typing**]
-   Tailwind [**Styles**]
-   Zustand [**Global State**]
-   Color accessibility
-   Prettier [**Code Formatter**]
-   Async Storage [**Persist Local Data**]
